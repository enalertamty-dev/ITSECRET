Instrucciones

  1. Escribe tu mensaje secreto en el campo de texto
  2. Opcionalmente adjunta un archivo o ingresa una URL
  3. Haz clic en "Generar Mensaje Cifrado"
  4. Se descargará un archivo HTML que puedes compartir
  5. El mensaje se autodestruirá después de ser visto una vez

Tus mensajes están protegidos

Utilizamos cifrado Base64 y almacenamiento local para garantizar que tus mensajes solo puedan verse una vez.

¿Cómo funciona?

Cuando alguien te envía un mensaje cifrado, recibirás un archivo HTML especial.

Para ver el mensaje:

  1. Selecciona el archivo usando el botón de arriba
  2. El mensaje se mostrará por 5 segundos
  3. Después de ser visto, se autodestruirá y no podrá verse again

Advertencia de seguridad

Solo abre archivos de personas en las que confíes. Nunca abras archivos HTML de remitentes desconocidos.
